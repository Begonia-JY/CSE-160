# WebGL Paint (Assignment 1)

This is a simple WebGL paint program where users can draw shapes on a canvas.

## Features
- Draw squares, triangles, and circles
- Change color with RGB sliders
- Adjust size and circle segments
- Clear canvas
- Draw a custom picture (bunny holding a carrot with C and L on ears)

## Files
- `index.html` – UI and canvas
- `coloredPoints.js` – main logic
- `Point.js` – square shape
- `Triangle.js` – triangle shape
- `Circle.js` – circle shape

## Notes
Some helper functions and the custom picture were developed with ChatGPT assistance.
