# Blocky Sloth (Assignment 2)

This is a simple WebGL 3D sloth made with cubes and matrix transformations.

## Features
* 3D animal made from cubes
* Global rotation slider
* Two sliders for joint control
* One joint chain (upper → lower → claw)
* Animation on/off
* Basic colors
* FPS display

## Files

* `index.html` – UI and canvas
* `index.js` – main logic
* `Cube.js` – cube drawing
* `lib/cuon-matrix.js` – matrix functions
* `lib/cuon-utils.js` – shader setup

## Notes

The part making the joints move correctly using matrices and keeping parts connected were done with ChatGPT help.
